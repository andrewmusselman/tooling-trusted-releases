Candidate drafts

A candidate draft is an editable set of files which can be frozen and promoted into a candidate release for voting on by the project's committee.

  • You can only create a new candidate draft if you are a member of the project's committee
  • Projects can work on multiple candidate drafts for different versions simultaneously
  • A candidate draft is only editable until submitted for voting
