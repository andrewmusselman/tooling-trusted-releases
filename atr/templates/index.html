Apache Trusted Release

ATR is a release management platform for Apache Software Foundation projects. It provides a standardized workflow for PMC members to submit, verify, and track release candidates.

Key Features

  • Automated PMC roster synchronization with Whimsy
  • Release candidate tracking and state management
  • Secure key management for release managers
  • Signature verification for release artifacts

Getting Started

To submit a release candidate, you must be a PMC member of the target project. First, add your signing key, then create a release candidate.
