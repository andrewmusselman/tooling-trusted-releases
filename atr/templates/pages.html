ATR Pages

A complete list of all pages and endpoints available in ATR.

Main Pages

/

Simple welcome page with Apache Trusted Releases title.
Access: Public

/pages

List of all pages on the website (this page).
Access: Public

PMC Management

/pmc/directory

Main PMC directory page with all PMCs and their latest releases.
Access: Public

/pmc/list

List all PMCs in the database (JSON format).
Access: Public

/pmc/<project_name>

Get details for a specific PMC (JSON format).
Access: Public

/pmc/create/<project_name>

Create a new PMC with sample data (JSON format).
Access: Public

Release Management

/add-release-candidate

Add a release candidate to the database.
Access: Committer
Additional requirement: Must be PMC member of the target project

/user/uploads

Show all release candidates uploaded by the current user.
Access: Committer

User Management

/user/keys/add

Add a GPG public key to your account for signing releases.
Access: Committer

/user/keys/delete

Debug endpoint to delete all GPG keys associated with your account.
Access: Committer
Warning: This is a debug endpoint that immediately deletes all your keys without confirmation!

Administration

/admin/database

Browse all records in the database.
Access: Committer Admin
Additional requirement: Must be in ALLOWED_USERS list

/admin/update-pmcs

Update PMCs from remote, authoritative committee-info.json.
Access: Committer Admin
Additional requirement: Must be in ALLOWED_USERS list

Debug/Testing

/secret

Test endpoint for authentication.
Access: Committer

/database/debug

Debug information about the database.
Access: Public
